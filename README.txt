PotatoTool by borygo#0001



Prosty tool do usnkrs, ale nie tylko

Instalacja:

Wypakuj exe do jakiegoś folderu i otwórz program. Przy pierwszym uruchomieniu trzeba program zrestartować, ponieważ tworzy potrzebne pliki.

Instrukcja:

1. Tworzy nazwy profili nie trzeba więcej się rozpisywać wpisujesz swoje imie lub nick i ile chcesz utworzyć kont, klikasz enter i program tworzy nazwy za ciebie i wystarczy je skopiować

2. Separator maili i hasła (potrzebne do linkowania kont w usnkrs) wklejasz konta do pliku txt accs i zapisujesz plik (Muszą być w formacie mail|haslo). Później odpalasz program i wybierasz mode nr 2 program usuwa wszystko, co znajduję sie za znakiem | (w pliku txt nic się nie zmienia potrzebne jest to tylko po to, aby program mógł wziąść sobie skąś dane), kopiujesz i wklejasz gdzie potrzebujesz.

3. Hashtag jig style, czyli jig w takim stylu superulicaxg, superulicawt itp. Wpisujesz nazwe ulicy i ile zjigowanych adresów chcesz dostać. Uwaga przy większej ilości adresów mogą być powtórki!

4. Double letters jig style, czyli tak jak sama nazwa wskazuje podwójne litery ssuperulica,suuperulica itp tworzy tyle adresów ile jest liter w głównym adresie

5. Tworzy randomowe numery domów. Ustawiamy zakres początkowy i końcowy. Uwaga przy mniejszej ilości numerów i małym zakresie mogą być powtórki!

6. Tworzy losowe polskie numery. Oczywiście one nie działają

7. Ctrl + c wychodzisz z programu
